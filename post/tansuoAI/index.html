探索AI | New Bing 能够识别本地PDF文档？！够强

· 2023-02-16 · # 探索

🐬收到必应内测邮件时也没有如此欣喜，像发现了新大陆一样高兴
🐌AI作为一项高智能的工具，还是需要我们去训练它的~

使用New Bing也好几天时间了，一开始以为这款“必应+ChatGPT”的新AI，可能也只是在基础聊天AI的基础上融入了更智能便捷的搜索功能（标注链接，触达全网），但没想到它能做的事情有点超乎我的想象（可能是我没接触到），它另有乾坤！超级强大！

缘起是在刷博主@Simon_阿文的动态时看到有关于微软演示New Bing的动图，我就想着去试验一下，因为只是匆匆一眼瞄了那个动图，以为是必应AI能够识别用户正在浏览的网页并且对网页内容进行解析。尝试好多网页发现识别失败，我还发了一条微博吐糟它没想到小丑竟然是我自己，我在推特上又刷大了那个动图，仔细看了一下，必应AI能够解析的是“在浏览器打开的PDF文档”

于是火速用浏览器打开了一篇崔庆龙的文章：http://mtw.so/5D7NRY，进行实验。坦白说一开始并不顺利，反反复复让必应AI回答终于解锁了这个超强功能：解析文档，可以针对文档内容对AI进行提问，让AI总结文档内容，归纳观点，分析观点，拓展观点等等（目前摸索到这些，其他未知项今后再探索吧）

微软演示的那个动图，描述的是：The GPT browser can understand and summarize a 15-page PDF in seconds。You can now ask for the the key takeaways of each page and chat about the content of the document.

就真的，真的很强悍了。如果能够恰当的把它运用到生活工作学习中，效率翻倍了吧。这款New Bing 真的深得我心~
